Software

Bioinformatics Software [GitHub].

GRO-seq Analysis Package. Powerful R package for the analysis of GRO-seq data. Automates many common next-generation sequencing data analysis tasks. Advanced features include a general HMM implementation useful for data segmentation (e.g. transcription unit identification, and Pol II wave calling). Currently under submission to Bioconductor.

AffyTiling [Bioconductor]. Easy extraction of individual probes in Affymetrix tiling arrays.
